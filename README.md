# Kata

## 
To run it :
- please connect to a local or online MongoDb Service, by modifying ` .backend/config/keys.js ` file.
- install NodeJS 

## Run Client
In `./client/` run `npm install` then  `npm start`.

## Run Server
In `./backend/` run `npm install` then  `node server.js`.
